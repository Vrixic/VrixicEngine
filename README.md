# VrixicEngine

Build / Launching Steps

*** Windows Platform ***

Step 1). Clone the repo / download main branch <br />
Step 2). Double click MakeProjectBuild.bat <br /> 
Step 3). Open Build/VrixicEngineBuild/VrixicEngine.sln <br />
Step 4). Build / Run the project (F5) <br />
